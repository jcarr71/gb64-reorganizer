## GameBase64 Game Organizer v1.3.0

### Bug Fixes
- Fixed --keep-zipped zip filename issue: Now uses game name instead of genre when creating zip files
- Type safety improvements: Added None checks for publisher and language fields to prevent attribute errors

### Features
- Dynamic template system with 15 metadata field placeholders
- English-only game filtering
- Publisher name simplification (--collapse-publishers)
- Keep-zipped option to archive games without extraction
- Automatic disk file renaming (D64, D71, D81, etc.)
- Duplicate handling with automatic versioning
- Comprehensive error logging

### Documentation
- Complete README with all features and examples
- Version management guide
- Command-line reference

### Downloads
- GameBase64Organizer.exe (9.73 MB): CLI version - recommended for automation
- GamebaseGameOrganizerGUI.exe (11.02 MB): GUI version (experimental)

### Example Usage
# Interactive mode
python gb64_reorganizer.py

# With options
python gb64_reorganizer.py "C:\Source" "C:\Dest" --template "{publisher}/{name}" --english-only --collapse-publishers

# Keep zipped
python gb64_reorganizer.py "C:\Source" "C:\Dest" --keep-zipped

### Available Template Fields
{name}, {primary_genre}, {secondary_genre}, {language}
{published_year}, {publisher}, {developer}
{players}, {control}, {pal_ntsc}
{unique_id}, {coding}, {graphics}, {music}, {comment}
